Retail Sales Data Agent on Databricks
Project Overview

This project demonstrates the creation of a Retail Sales Data Agent using Databricks AI/BI Genie. The agent enables business users to ask natural language questions and receive insights from retail sales data.

Dataset

The dataset contains 1,000 retail transactions including:

Transaction ID
Date
Customer ID
Gender
Age
Product Category
Quantity
Price per Unit
Total Amount
Tools Used
Databricks
SQL
AI/BI Genie
GitHub
Key Results
Total Revenue: R456,000
Highest Revenue Category: Electronics
Best Sales Month: May 2023
Highest Spending Segment: Customers Under 30
Top Revenue Contributor: Female Customers
Sample Questions
What is total revenue?
Which category performs best?
Which month generated the highest sales?
What are the sales trends?
Which customer segment spends the most?
Business Recommendations
Invest more in Electronics inventory.
Target younger customers through digital campaigns.
Expand loyalty programs.
Prepare stock levels before peak months.

Conclusion
The Retail Sales Data Agent successfully answers business questions using natural language and provides actionable retail insights.
